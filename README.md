## Tour por el código
Esto es el orden de actuación de mi código:
1º Genero el fichero aleatorio
2º Comparo linea por linea de ese fichero con el de nuestro diccionario para encontrar coincidencias
3º Este a su vez va generando en una variable String todo el contenido con etiquetas <strong> para señalar donde están las coincidencias y en qué línea/s
4º Volcado de todo el contenido a resultados.html
